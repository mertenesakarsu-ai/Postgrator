# Backups Directory

Bu klasör .bak dosyalarınız için kullanılır.

## Kullanım

.bak dosyalarınızı buraya koyabilirsiniz veya uygulama üzerinden yükleyebilirsiniz.

## Not

- Bu klasör Docker container'ı ile paylaşılır
- Büyük dosyalar için yeterli disk alanı olduğundan emin olun
- Git'e commit edilmez (.gitignore'da tanımlı)
